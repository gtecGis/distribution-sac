Install SAC

Important: Certificate Installation Required

Before installing SAC, you must install our security certificate. This is a one-time process to ensure secure app installation.

1

Download the Certificate

First, download our security certificate:

Download Certificate
2

Install the Certificate

Double-click the downloaded certificate file and follow these steps:

  1. Click "Open" or "Run" when prompted
  2. Click "Install Certificate"
  3. Select "Local Machine" and click "Next"
  4. Choose "Place all certificates in the following store"
  5. Click "Browse" and select "Trusted People"
  6. Click "Next" and then "Finish"
  7. Click "Yes" on the security warning
